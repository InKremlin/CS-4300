# Explanation

In this project I decied to explore the orientation option for additional reaction effects. The Karl Sims description was: "diffusion can occur faster in one direction than another to give an orientation to the results." At first I tried doing some trigonemetry to make a controllable slider that allowed the user to change the diffusion bias from a 0-360 degrees, but this ended up being too complicated. Instead I went for a vertical bias. I had some issues with the lapacian weights, when unbalanced they would cause the simulation to create visual artifacts like solid black cubes. I eventually setlled on some values I was ok with, the elongated pattern reminded me a lot of snowflakes. I allowed the user to adjust the diffusion rates, feed and kill values, to reset the simulation.
